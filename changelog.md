## [0.0.2] - 1/22/26

### Added

- If statements (Else, else-if, not supported yet)
  - if statements are created with the keyword: **comparator**. 
  - Example: 
    ```rsd
    comparator (on){
        chat("Hello World!")
    }
    ```
